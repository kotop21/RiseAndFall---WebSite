# Параметры запуска игры

Здесь описаны ключевые параметры запуска, которые могут использоваться для управления поведением игры при старте.

---

## 📁 `-datapath`

**Описание:**
Указывает путь к папке с игровыми ресурсами.

**Синтаксис:**

```
-datapath "<имя_папки>\"
```

**Особенности:**

* Параметр **не чувствителен к регистру** (`-datapath`, `-DataPath`, `-DATAPATH` и т.д.)
* Путь должен **заканчиваться символом `\`**

**Пример:**

```
-datapath "Data\"
```

---

## 📦 `-redistpath`

**Описание:**
Указывает путь к папке с необходимыми библиотеками игры.

**Синтаксис:**

```
-redistpath "<имя_папки>\"
```

**Особенности:**

* Путь должен заканчиваться символом `\`

**Пример:**

```
-redistpath "redist\"
```

---

## 🚫 `-nodump`

**Описание:**
Отключает создание crash-логов при сбоях игры.

**Синтаксис:**

```
-nodump
```

**Особенности:**

* Не принимает аргументов

---

## 🔄 `-forceautodetect=1`

**Описание:**
Принудительно сбрасывает настройки игры при каждом запуске.

**Синтаксис:**

```
-forceautodetect=<значение>
```

**Особенности:**

* Работает при любом значении, **кроме `0`**
* При каждом запуске выполняется полный сброс настроек
* После сброса параметр необходимо удалить, иначе настройки не будут сохраняться

**Пример:**

```
-forceautodetect=1
```

---

## ⚠️ Примечания

* Все пути рекомендуется указывать в кавычках
* Ошибки в путях могут привести к некорректной работе игры
